Malva Gård

The project concerned the redesign of the web page for the Swedish company Malva Gård. The goal was to improve the UI by making the web page easier to navigate through, less dense information, and with a better flow. I used HTML and CSS for the markup and Affinity designer and Sketch for the Graphic design. The project was carried out January 2017 - April 2017.
